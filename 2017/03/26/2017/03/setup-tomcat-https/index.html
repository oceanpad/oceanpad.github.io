Setup Tomcat Https

本教程使用 JDK 6 和 Tomcat 7，其他版本类似。

基本步骤：

  • 使用 java 创建一个 keystore 文件
  • 配置 Tomcat 以使用该 keystore 文件
  • 测试
  • 配置应用以便使用 SSL ，例如 https://localhost:8443/yourApp
